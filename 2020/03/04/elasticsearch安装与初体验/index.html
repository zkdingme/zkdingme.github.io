docker 安装 elk

已有前人整理好在github https://github.com/deviantony/docker-elk

elk版本可在.env文件中修改，20200325下载时.env中版本号为7.6.0

当前默认用的是30天试用版本，可以在es配置文件es.yml中修改trial为basic改成免费版

The stack is pre-configured with the following privileged bootstrap user:

  • user: elastic
  • password: changeme

docker-compose up 拉取镜像并生成容器

docker-compose down -v 删除容器、镜像和数据

插入数据可以在elk都配好之后

cat data.csv | nc -c localhost 5000 通过logstash插入到es中

可以在logstash配置文件logstash.conf中配置fliter来将原始数据变成结构化数据

然后kibana打开
